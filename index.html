Automated Malware Detector (AMD-IDS)

Real-Time Framework for Behavioral Analysis (In Progress)

Project Overview

This project is focused on developing a real-time framework for detecting and analyzing malicious file behavior through virtual machine analysis and machine learning. By integrating Elasticsearch, Logstash, Kibana, Sysmon, and Beats, we aim to predict malicious activity based on dynamic behavioral analysis, providing real-time insights and detecting potential threats effectively.
