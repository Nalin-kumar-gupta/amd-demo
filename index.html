Malicious File Detection and Dynamic Analysis

Work in Progress - Real-Time Framework for Behavioral Analysis

Project Overview

This project focuses on developing a real-time framework for detecting and analyzing the behavior of malicious files using virtual machines. By utilizing Cuckoo Sandbox, Docker, and decision tree models, we are able to predict malicious activity through dynamic behavioral analysis. The goal is to provide real-time insights into file behavior and detect potential threats.
